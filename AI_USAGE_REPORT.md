# AI_USAGE_REPORT

# AI Usage Report for JARVIS AI Assistant

## AI Tools Used

* ChatGPT
* Local AI Model (via Ollama/local inference)

---

# Purpose of Using AI

AI was used as a development assistant to accelerate coding, debugging, UI design, and feature implementation. The project was developed using an iterative approach where AI-generated code was tested, modified, and integrated manually.

---

# Major Areas Where AI Was Used

## User Interface Design

AI assistance was used to:

* Design the cyber-themed JARVIS interface.
* Create the three-panel layout.
* Implement CPU, RAM, and Battery progress bars.
* Create the conversation area.
* Design weather and network sections.
* Improve alignment and spacing.

# Challenges Faced

During development, several challenges were encountered:

* Tkinter and threading conflicts.
* Camera integration issues.
* OpenCV frame updates.
* UI alignment problems.
* Synchronization between speech recognition and GUI.
* Face detection integration.
* Handling background tasks without freezing the interface.

The following tasks were performed manually:

* Project planning.

* Debugging generated code.
* Modifying and improving AI-generated code.

* UI refinement.

# Conclusion

AI significantly accelerated development by assisting with coding, debugging, and design. However, the final architecture, integration, testing, and refinement of the system were performed manually. This project demonstrates how modern developers can effectively collaborate with AI tools to build practical and feature-rich applications.
